# Лабораторная работа №3 (уровень 2)
**Моделирование знаний с НЕ-факторами**

## Что реализовано

В папке реализована экспертная система по медицинской диагностике на Prolog:

- `knowledge_base.pl` — база знаний:
  - 12 симптомов
  - 5 диагнозов
  - веса симптомов для каждого диагноза
  - пары несовместимых диагнозов
- `inference.pl` — вывод и валидация:
  - ввод наблюдений (`set_observations/1`)
  - вычисление баллов и уверенности (`diagnosis_score/2`, `diagnosis_confidence/2`)
  - объяснение вывода (`explain/2`)
  - проверка полноты БЗ (`coverage_issues/1`)
  - проверка непротиворечивости (`is_consistent/0`)
- `tests.pl` — 5 тестовых сценариев для демонстрации.

## Соответствие уровню 2 из методички

- Используется NAF (`\+ observed(S)`) для работы с отсутствующими симптомами.
- Есть объяснение вывода (какие симптомы сработали/отсутствуют).
- Реализована валидация БЗ:
  - полнота покрытия симптомов;
  - непротиворечивость активных гипотез.

## Как запускать

```prolog
?- [tests].
?- run_all_cases.
```

Отдельные запросы:

```prolog
?- set_observations([температура,кашель,потеря_вкуса,одышка]).
?- best_diagnosis(D, Score, Confidence).
?- explain(covid19, E).
?- coverage_issues(U).
?- is_consistent.
```
